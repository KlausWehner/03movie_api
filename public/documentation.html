MOVIE API

This is a server-side component of a “Movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

LIST OF MOVIES

GET /movies
Returns a list of all movies in the database

REQUEST HTTP METHOD ENDPOINT URL REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Get a list of all movies GET /movies NONE JSON object with list of ALL movies available on the datatbase
Return available data details about a single movie by title to the user GET /movies/[Title] NONE JSON object with description, genre, director and image URL.
Return movies by type of genre by genre name (e.g., “Thriller”) GET movies/Genre/[name] NONE Return description of the requested genre
Return Director's Bio GET movies/Director/[name] NONE Return JSON object containing a short bio about the director.
Register a new user POST /users A JSON object holding data about the new user, structured like: { Username: "", Email: "", Password : "", Birthday: "", } A JSON object holding data about the new user, including an ID { _id: "", Username : "", Email: "", Password : "", Birthday: "" }
Login as a registered user POST /login/[Username]
User to update their own info PUT /users/[Username] NONE An updated JSON object holding altered data about the user, including the assigned ID { _id: Integer, Username : "", Email: "", Password : "", Birthday: "" }
Add Movie to favorites List POST /users/[Username]/favorites/[movieID] NONE Creates a favourite list. Showing a text that a movie had been added
Remove Movie from Favorites List DELETE /users/[Username]/favorites/[movieID] NONE Removes a movie from the users favorites list, Showing a text that a movie had been removed
Delete account DELETE /users[Username] NONE Allows a user delete their account
